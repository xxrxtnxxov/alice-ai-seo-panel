# Панель SEO для Алисы AI

Userscript для анализа ответов Яндекс Алисы в поисковой выдаче. Отслеживает источники, домены, позиции в органике и ведёт журнал запросов.

![License](https://img.shields.io/badge/license-MIT-green)

## 📥 Установка

### Через Tampermonkey (рекомендуется)

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для Chrome/Firefox/Edge
2. Нажмите на значок Tampermonkey → **"Создать новый скрипт"**
3. Удалите весь код и вставьте содержимое файла `script.userscript.js`
4. Нажмите **Файл → Сохранить** (Ctrl+S)
5. Откройте https://yandex.ru/search/?text=запрос — панель появится автоматически (если есть блок Алисы)

### Прямая ссылка для обновлений

- **Raw URL**: https://raw.githubusercontent.com/xxrxtnxxov/alice-ai-seo-panel/main/script.userscript.js
- Добавьте скрипт как URL в Tampermonkey для автоматических обновлений

## ✨ Возможности

### 📊 Панель анализа

- **Обзор** — текущий запрос, наличие блоков Алисы слева/справа, тип ответа (текст/список/таблица/структурированный)
- **Источники** — домены, которые Алиса использовала для ответа (левая и правая колонка раздельно)
- **Пересечение с органикой** — какие источники Алисы также ранжируются в обычной выдаче
- **Люди ищут** — похожие запросы из блока "Люди ищут"
- **Скопировать ответ** — копирует текст ответа Алисы с форматированием (заголовки, списки)

### 🎯 Мой сайт

- Добавьте ваш домен — скрипт отслеживает его присутствие в источниках
- Подсвечивает ваш домен зелёным в чипах источников
- Показывает позицию в органической выдаче

### ⚔️ Конкуренты

- Добавьте домены конкурентов — они подсвечиваются оранжевым
- Отслеживаются в журнале и статистике

### 🗃 База данных

- **Журнал запросов** — история всех запросов с отметками о присутствии вашего сайта
- **Топ доменов** — статистика частоты появления доменов в источниках Алисы

### 💾 Экспорт

- CSV-экспорт журнала, источников, статистики
- Копирование в буфер обмена
