# 🤖 Панель SEO для Алисы AI

> UserScript для Tampermonkey / Violentmonkey — SEO-анализ блоков Алисы AI прямо на странице поиска Яндекса.

![alice.jpg](alice.jpg)

---

## Что делает скрипт

Скрипт встраивает в правый угол страницы `yandex.ru/search` плавающую панель, которая в реальном времени анализирует блоки **Алисы AI** — как в левой колонке (тизер нейроответа), так и в правой (карточка).

### Возможности

- **Обзор выдачи** — тип ответа Алисы (текст / список / таблица / структурир.), число источников, процент первого экрана, занятый тизером
- **Источники** — чипы с доменами и ссылками прямо над текстом ответа; экспорт доменов и URL в буфер или CSV
- **Мой сайт** — укажите свои домены (поддержка `*`-масок), скрипт подсветит зелёным где они есть и отметит ❌ где нет
- **Конкуренты** — аналогично, но оранжевым; видно кто попал в источники вместо вас
- **Органика vs источники** — пересечение доменов Алисы с обычной выдачей с указанием позиций
- **Люди ищут** — список похожих запросов из блока «Люди ищут» с экспортом в CSV / буфер
- **Копировать ответ** — извлекает чистый текст ответа Алисы с сохранением структуры (заголовки, списки)
- **Журнал запросов** — история всех поисков с отметками о присутствии вашего сайта, до 500 записей; экспорт в CSV
- **Топ доменов-источников** — накопительная статистика, какие домены чаще всего попадают в источники
- **Экспорт / Импорт** — бэкап и перенос всех настроек и данных через JSON-файл
- **Перетаскивание** — панель можно переместить в любое место экрана, позиция сохраняется

## Установка

1. Установите расширение для браузера:
   - [Tampermonkey](https://www.tampermonkey.net/) (Chrome, Firefox, Edge, Safari)
   - [Violentmonkey](https://violentmonkey.github.io/)

2. Нажмите на ссылку ниже — расширение откроет диалог установки:

   **[⬇ Установить скрипт](https://github.com/xxrxtnxxov/alice-ai-seo-panel/raw/refs/heads/main/script.userscript.js)**

3. Подтвердите установку и перейдите на [yandex.ru/search](https://yandex.ru/search/?text=seo) — панель появится автоматически.

---

## Быстрый старт

1. Введите любой запрос в Яндексе
2. Если Алиса показала блок — панель отобразит данные автоматически
3. В разделе **🎯 Мой сайт** введите ваш домен (например `site.ru`) и нажмите `＋`
4. Перезагрузите страницу — скрипт будет подсвечивать ваш сайт в источниках

### Цветовая кодировка источников

| Цвет | Значение |
|---|---|
| 🟦 Синий | Источники левой колонки (тизер) |
| 🟣 Фиолетовый | Источники правой колонки (карточка) |
| 🟢 Зелёный | Ваш домен |
| 🟠 Оранжевый | Домен конкурента |

## Журнал запросов

Каждый поиск сохраняется автоматически. В журнале отображается:

| Столбец | Описание |
|---|---|
| Дата | Дата запроса |
| Запрос | Поисковый текст |
| Мой сайт | ✅ есть / ✗ нет / — блок отсутствует (лев. \| прав.) |
| Тип ответа | текст, список, таблица и т.д. |
| Источников | кол-во источников лев. / прав. |
| Экран | % первого экрана, занятый тизером |

Журнал хранит до **500 записей**. Экспорт в CSV доступен прямо из модального окна.

## Технические детали

- **Версия:** 4.2.4
- **Платформа:** `https://yandex.ru/search*`
- **Хранение данных:** `GM_setValue` / `GM_getValue` (локально в браузере)
- **Используемые GM-функции:** `GM_setValue`, `GM_getValue`, `GM_setClipboard`, `GM_registerMenuCommand`, `GM_download`
- **Обнаружение блоков:** `MutationObserver` + множественные CSS-селекторы с fallback-цепочками (без хрупких генерируемых классов)
- **Обновление скрипта:** автоматически через Tampermonkey (updateURL прописан в заголовке)
